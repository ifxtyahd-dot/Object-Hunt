# Object-Hunt
# 📡 ANCHOR NET // MOBILE OPERATIVE GRID

A lightweight, high-immersion, real-world location game built entirely in HTML5, JavaScript, and Tailwind CSS. Players locate physical tracking items hidden across real-world coordinates, perform a matching verification verification check, and plant new assets to keep the decentralized loop alive.

## 💰 Integrated Monetization Channels
* **Energy Capacitor Gating:** Limits standard free accounts to 2 operations per cycle.
* **Rewarded Ad Simulation Engines:** Allows users to opt-in to ad-payloads to instantly regenerate energy points.
* **Premium Operative License Tier:** Upgrades accounts to $4.99/mo premium clearance, dropping all energy ceilings and unlocking hidden high-tier coordinate zones.

## 🚀 How to Deploy (Completely Free)
1. Fork or copy this repository.
2. Link the repository to **Netlify** or **Vercel**.
3. Launch your live generation build. It is fully optimized out-of-the-box for mobile screen views.
